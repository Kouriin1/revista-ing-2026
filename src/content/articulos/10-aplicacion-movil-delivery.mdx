# Aplicación Móvil para la Gestión de Delivery y Rutas Óptimas

## Objetivo Principal

Desarrollar una solución móvil multiplataforma que integre algoritmos de geolocalización y optimización de rutas (Problema del Viajante - TSP) para mejorar los tiempos de entrega de mercancía de la microempresa Rapitienda.

## Metodología

Investigación tecnológica encuadrada bajo la metodología de desarrollo de software AUP (Agile Unified Process).

## Población y Muestra

Análisis geográfico de 120 zonas de entrega frecuentes en el área metropolitana de Caracas y un grupo piloto de 8 repartidores motorizados.

## Técnicas de Análisis

Simulación de rutas comparando el algoritmo tradicional (elección empírica del motorizado) versus el Algoritmo de Dijkstra modificado con Google Directions API.

## Resultados y Pruebas Métricas

En las pruebas de simulación logística, el uso de las rutas optimizadas redujo la distancia recorrida en un 27,4% promedio por despacho. Esto se tradujo directamente en un ahorro del 19,5% en gastos de combustible mensual para la flota y una baja del tiempo de espera del cliente final de 45 a 31 minutos.

## Propuesta Tecnológica

Aplicación desarrollada bajo el framework **Angular** con arquitectura **Ionic** para distribución nativa en Android e iOS. El backend corre bajo servicios serverless en **AWS (Amazon Web Services)** utilizando DynamoDB como base de datos NoSQL para aguantar alta concurrencia de rastreo geográfico en tiempo real.
