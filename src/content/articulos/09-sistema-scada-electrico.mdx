# Sistema SCADA para el Monitoreo Eléctrico

**Tutor:** Ing. Nelson Rivas.

## Objetivo Principal

Diseñar un sistema de Supervisión, Control y Adquisición de Datos (SCADA) de arquitectura abierta para optimizar el monitoreo de variables eléctricas en los transformadores principales de la subestación de distribución eléctrica interna del complejo universitario USM.

## Metodología

Investigación aplicada, de carácter descriptivo y diseño experimental de campo.

## Variables Monitoreadas

Corriente (A), Voltaje (V), Factor de Potencia y Temperatura del aceite térmico de enfriamiento.

## Instrumentación

Analizadores de red con comunicación serial Modbus-RTU y transductores de temperatura PT100.

## Resultados del Diagnóstico

Se identificaron picos de sobrecarga severos no registrados durante las horas del mediodía que acortaban la vida útil de los equipos aislantes. El sistema actual dependía exclusivamente de revisiones analógicas manuales una vez por semana, imposibilitando la acción preventiva.

## Propuesta Tecnológica

Interfaz de usuario (HMI) desarrollada sobre el software de supervisión industrial Ignition SCADA. El enlace de campo se realiza mediante un gateway industrial que convierte los datos Modbus-RTU a MQTT para transmisión inalámbrica de largo alcance. El sistema emite alertas criticas automatizadas vía SMS y correo electrónico cuando los parámetros sobrepasan los limites seguros establecidos.
